# vue-edu

### Compiles and hot-reloads for development
```
npm run serve
```

### 开启本地服务
```
cd mock
node server.js
```

### 技术点

1. 用vue-router作前台路由
2. 用`axios`发送`ajax`请求数据
3. 用`Node`做后台服务，自定义模拟数据
4. 引入`vue-awesome-swiper`插件做轮播
5. 原生`scroll`事件做上拉加载
6. 渲染老师列表用到瀑布流布局
